# ✨ AI 写真馆

一键生成你的专属 AI 写真，支持多种风格切换。

## 功能

- 📸 上传正面照片
- 🎨 6种写真风格可选（日系清新、韩式证件照、复古港风、赛博朋克、国风古韵、杂志封面）
- ✨ AI 一键生成
- 📥 下载生成结果

## 技术栈

- 前端：HTML + CSS + JavaScript
- 后端：Node.js
- AI：AnyCap API (GPT Image 2)

## 快速开始

```bash
# 启动服务
node server.js

# 打开浏览器
http://localhost:3000
```
